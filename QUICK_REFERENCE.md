# Quick Reference Guide

Fast reference for all scoreboard operations matching the Daktronics AllSports 4100 manual.

## üèí Team Names

### How to Change Team Names (Per Manual)

**From Manual: "To Display Team Names"**

1. **Enter Menu**
   - Press MENU button (black)
   - Status shows: `MENU`

2. **Select Team Name Mode**
   - Scroll down to "Team Name Mode"
   - Press ENTER
   - Team Name overlay appears

3. **Press POST**
   - Status shows: `Press POST, then LEFT/RIGHT to choose team`

4. **Select Team (LEFT or RIGHT)**
   - Press LEFT button = HOME team
   - Press RIGHT button = GUEST team
   - Keyboard overlay is active

5. **Type Team Name**
   - Press letter/number buttons on overlay
   - Press SPACE for spaces
   - Press BACK SPACE to delete
   - Press CLEAR to reset name
   - Max 12 characters

6. **Save Name**
   - Press ENTER when done
   - Status shows: `Team name saved`

7. **Edit Other Team (Optional)**
   - Press POST again
   - Press LEFT or RIGHT for other team
   - Type new name, press ENTER

8. **Exit Team Name Mode**
   - Press ESC or EXT button
   - OR press MENU to exit

### Quick Workflow

```
MENU ‚Üí Team Name Mode ‚Üí ENTER ‚Üí POST ‚Üí
LEFT (or RIGHT) ‚Üí Type name ‚Üí ENTER ‚Üí
ESC to exit
```

2. Type player number (e.g., `14`)
3. Press ENTER
4. Type time (e.g., `200` for 2:00) OR press ENTER for default

2. **Navigate to penalty** (use ‚Üë/‚Üì arrows)

## üéÆ Keyboard Overlay (Team Names)

The overlay appears when in Team Name Mode:

```
Row 1: LEFT  RIGHT  DOUBLE  SINGLE  DM  ESC  POST  EXT
Row 2: Q  W  E  R  T  Y  U  I  O  P
Row 3: A  S  D  F  G  H  J  K  L  .
Row 4: Z  X  C  V  B  N  M  SPACE  BACK-SPACE  #
```

**Special Buttons:**
- **LEFT** = Select HOME team
- **RIGHT** = Select GUEST team
- **POST** = Confirm team selection
- **ESC/EXT** = Exit team name mode
- **SPACE** = Add space
- **BACK SPACE** = Delete last character
- **#** = Message number character

## ‚ö° Important Notes

### Penalties
- ‚úì Penalties only count down when game clock is running
- ‚úì Auto-remove when reaching 0:00
- ‚úì Can have unlimited penalties (shows top 2 per team)
- ‚úì Use CLEAR (no buffer) to manually remove

### Team Names
- ‚úì Max 12 characters
- ‚úì Uppercase only (A-Z, 0-9, space, period, #)
- ‚úì Must enable overlay first (checkbox or Team Name Mode)
- ‚úì Changes visible immediately on scoreboard

### Clock
- ‚úì Default period time: 20:00
- ‚úì Auto-plays horn at 0:00 in countdown mode
- ‚úì Penalties pause when clock stops
- ‚úì END button also exits menus

## üîß Troubleshooting

**Team name overlay not showing?**
‚Üí Click MENU ‚Üí Team Name Mode ‚Üí Press POST

**Can't type team name?**
‚Üí Make sure you pressed POST, then LEFT or RIGHT

**Penalty won't remove?**
‚Üí Make sure no numbers are typed, just press CLEAR

**Clock won't start?**
‚Üí Check if already running, press END to stop first

**Penalties not counting down?**
‚Üí Make sure game clock is running (START button)
